SimplePCTools
SimplePCTools is my first PC application, created to help with basic disk cleanup, simple internet speed checking, and easy photo management. This is an early beginner version, so bugs or limitations may appear. Feedback and suggestions are welcome to help improve future updates.

Features
Basic disk space cleanup

Quick internet speed check

Simple photo tools

Project Status
This is an initial learning project. More features, fixes, and improvements will be added over time.
